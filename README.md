This Tool is uses python 3.12 with the Spotdl Library to gather information from Spotify.

To use the tool:

Spotify:
1) Get the URL of your playlist, ensure your playlist is public
2) You will be prompted to make a folder to store your media in, but only if the default "Music" folder is not found.
3) The Music will begin to download
4) Check the folder in which it was downloaded and enjoy!
